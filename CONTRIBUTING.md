# Contributing to MudX

Thank you for your interest in contributing to **MudX** — a community-driven extension library built on top of [MudBlazor](https://github.com/MudBlazor/MudBlazor).

🚧 We currently follow the same contribution guidelines as the core MudBlazor project.

Please refer to their contributing guide here:

👉 [MudBlazor CONTRIBUTING.md](https://github.com/MudBlazor/MudBlazor/blob/dev/CONTRIBUTING.md)

---

We’ll expand this document with MudX-specific contribution tips and workflows as the project evolves.

Feel free to open a [Discussion](https://github.com/MudXtra/MudX/discussions) if you have questions or ideas!
